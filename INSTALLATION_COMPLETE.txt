   - Assurez-vous que Apache et MySQL sont démarrés dans Laragon

2. ACCÉDER AU JEU
   - Ouvrez votre navigateur
   - Allez sur : http://localhost/Mafia-Game-Script/

3. CONFIGURER LES PARAMÈTRES DU JEU
   - Allez dans la section Admin
   - Modifiez les paramètres du jeu selon vos préférences

4. CONFIGURER LES CRON JOBS (IMPORTANT !)

     * Tâches quotidiennes

DÉPANNAGE
---------

Si le site ne s'affiche pas :
1. Vérifiez que Laragon est démarré
2. Vérifiez que Apache et MySQL sont actifs
3. Testez l'accès à : http://localhost/

Si vous ne pouvez pas vous connecter :
1. Vérifiez que la base de données existe :
   mysql -u root -e "SHOW DATABASES LIKE 'mafia_game';"
2. Vérifiez que l'utilisateur existe :
   mysql -u root -e "USE mafia_game; SELECT * FROM users WHERE username='admin';"

Erreurs PHP :
1. Le script utilise du code obfusqué (install.php est encodé)
2. C'est normal, le jeu devrait fonctionner sans problème
3. Si vous voyez des erreurs, vérifiez les logs PHP dans Laragon
